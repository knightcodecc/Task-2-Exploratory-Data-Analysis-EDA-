# Titanic Dataset EDA

This project contains an exploratory data analysis (EDA) notebook for the Titanic dataset.

## Files
- Task2_EDA_Titanic.ipynb — Jupyter notebook with data loading, cleaning, statistics, and visualizations
- Titanic-Dataset.csv — Titanic dataset used in the analysis

## Requirements
Install the following Python packages if needed:
- pandas
- numpy
- matplotlib
- seaborn
- plotly
- nbformat

## How to Run
1. Open the notebook in Jupyter Notebook or VS Code.
2. Run the cells in order.
3. The notebook will load the dataset, explore the data, and generate visualizations.

## Objective
To understand the Titanic dataset using descriptive statistics and visual plots, including:
- missing values
- distributions
- correlations
- survival patterns by category
